# Task Tracker

### Варіант індивідуального завдання

Мій номер у групі (N) - 19. Тоді:

V2 = (N % 2) + 1 = (19 % 2) + 1 = 1 + 1 = 2

V3 = (N % 3) + 1 = (19 % 3) + 1 = 1 + 1 = 2

V5 = (N % 5) + 1 = (19 % 5) + 1 = 4 + 1 = 5

##### Опис застосунку(V3)

Task Tracker — сервіс для відстеження задач.

Об’єкт задачі має наступні поля:

- id
- title
- status
- created_at

API сервісу складається з 3 ендпоінтів:

- GET /tasks — вивести усі задачі (id, title, status, created_at)
- POST /tasks (title) — створити нову задачу
- POST /tasks/`<id>`/done — змінити статус задачі на виконано

Назва застосунку незалежно від варіанту mywebapp

##### Спосіб конфігурації (V2)

Конфігураційний файл за шляхом /etc/mywebapp/config.json
